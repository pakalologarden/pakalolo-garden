# pakalolo.garden — Canonical Direction

## Core Objective
Build a trusted cultivation community centered around Hawaiʻi,
regenerative growing, and real-world experience.

## Value Ladder
Knowledge → Trust → Community → Relationships → Consulting → Products → Services → Partnerships

## Primary Message
Join a growing community of Hawaiʻi cultivators, gardeners,
homesteaders, and curious people learning regenerative cultivation together.

## Button Language
- Join the Community
- Join the Ohana
- Stay Connected
- Follow the Journey

## Avoid
- "Growing Notes" overuse
- "Get Notes" / "Subscribe for Notes"
- Describing format instead of value
- Selling consulting directly

## The Value Is
- Connection and community
- Real-world Hawaiʻi experience
- Learning alongside someone doing the work
- Access to a knowledgeable local resource
- Following a long-term cultivation journey

## Every Page Should Answer
"Does this make someone want to join the community?"
